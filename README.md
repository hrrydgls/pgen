# PGen - Your password generator in your CLI

How to use:

```bash
pgen length [options]

# length is 8 by default and options are ulns

# Options:
# u -> upper case
# l -> lower case
# n -> numbers
# s -> symbols

```

You can even run it like this:

```sh
➜ ./pgen 20           
j3T72Y869XDM}8U9?d)v

➜ ./pgen 20 u
YGRMKFFNSPQFJHPWVNGZ

➜ ./pgen u   
TTVEXFTK

➜ ./pgen   
3/B{2NZe

```
